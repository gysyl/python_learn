# Python 基础 (Basics)

本模块专注于学习和实践 Python 的核心基础知识。

## 核心知识点

- 变量与数据类型 (str, int, float, bool)
- 常用数据结构 (list, tuple, dict, set)
- 控制流 (if-elif-else, for, while)
- 函数的定义与调用
- 模块与包的导入 (`import`)

## 如何运行

### 实践代码

```bash
python basics/main.py
```

### 单元测试

```bash
pytest basics/tests/
```
