# 数据科学入门

## 安装依赖与启动 Jupyter
- 激活虚拟环境：`./.venv/Scripts/Activate.ps1`
- 安装依赖：`pip install -r data_science/requirements.txt`
- 启动 Notebook：`jupyter notebook`

## 建议练习内容
- `01_pandas_basics.ipynb`：导入数据、索引筛选、聚合与合并。
- `02_cleaning_visualization.ipynb`：缺失值处理、类型转换、绘制 2–3 个图表。
- `03_ml_intro.ipynb`：训练一个分类或回归模型，评估与交叉验证。

## 产出
- 每个 Notebook 都输出一页报告（图或结论）。
- 形成一个通用模板，后续项目直接复用。
